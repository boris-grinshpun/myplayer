### Run in this order

1. CDN
- cd cdn
- npm install
- npm run cdn

2. Server
- cd server
- npm install
- node index.js / nodemon index.js 

3. Client
- cd client
- npm install
- npm run dev
---
### Tests
- cd client
- npm test
- npm test:coverage
